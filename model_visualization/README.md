# AI Audio Classification Dashboard

A production-grade research dashboard for audio CNN inference visualization.

## Features

- Audio upload & playback
- Waveform visualization
- Spectrogram display
- CNN feature map explorer
- Model confidence charts
- Modern glass UI

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS
- Recharts
- Framer Motion
- WaveSurfer.js

## Run Locally

npm install
npm run dev
